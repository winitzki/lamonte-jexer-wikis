TText
=====

TText is a simple scrollable text area. It reflows automatically on
resize.

-----------

![text_2](uploads/ca9640315db47d08110b71212e9aa262/text_2.png)

Examples
--------

```Java
textField = addText("This is an example...", x, y, width, height);
```
