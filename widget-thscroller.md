THScroller
==========

THScroller is a simple horizontal scroll bar.

![hscroller_1](uploads/bb0fe3a81ef2debcc2a6deb7efc59be5/hscroller_1.png)

Examples
--------

```Java
THScroller hScroller = new THScroller(this, 17, getHeight() - 2, getWidth() - 20);
```
